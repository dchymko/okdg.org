October HackNight — October 11th 2011

How to be Oauth-some by Daryl Chymko

In 10 minutes or less, Daryl will cover the Oauth process and show code examples of using it to connect to APIs such as Facebook, Twitter and Foursquare

Twilio 101 by Roger Stringer

Roger will give a quick overview on how to use Twilio to add SMS and voice functionality to your apps

OKDG Hack night

We’ve had members asking for a hacknight that includes a quick 'how to' session at the start with code examples. We’re happy to announce an API Hacknight on Nov 11th that will showcase how to use Twilio, Twitter and Foursquare in your applications. If you are looking at adding voice or SMS functionality to your apps or learning how to use Oauth, this night is for you!

The event is happening from 7-10pm at the recently-opened OK co+Lab. Bring a laptop and your ideas. We will begin the night be giving a technical overview and then spend the remaining building out apps. For this event we will be demoing mostly in PHP but will be able to help with other platforms such as .NET, Ruby, etc. There is no agenda, just come out to learn and lay.

It’s important that you arrive on time since the presentations need to be finished by 7:30 to ensure enough time for hacking.

Hope to see you there!
