Applications Software Developer

Software Modelers Inc in West Kelowna, BC

Skills & experience required:

·         Degree or diploma in Computer Science (or equivalent) 

·         Minimum 2 years application software development experience 

·         SQLServer 2005/2008

·         Formal data modeling (relational & object-oriented)

·         .Net Framework 3.5 or later

·         C#

·         Javascript

·         Object-Oriented Analysis & Design

·         ASP.Net (client & server-side coding)

·         VisualStudio 2008 or later

·         .Net application deployment (web or desktop)

 

·         Personal attributes:

o    Strong communication skills (both written and verbal)

o    Good organizational and time management skills

o    Able to work independently or within a team

o    Friendly but professional manner and attitude

Nice to have (any of the following):

·         Financial or gaming systems applications development

·         Multi-tiered/layered application architectures

·         Agile/Test-Driven Development (NUnit, MSTest, etc.)

·         Telerik RADControls (including AJAX & client-side coding)

·         WCF or Web-Services

·         HTML, CSS, XML 

·         C++ (object-oriented)

·         Java

·         jQuery, YUI or other Javascript toolkit

·         Design Patterns

·         Windows desktop application development (WPF, MFC)

·         iPhone or Win Mobile application development

The role:
We are looking for an experienced application software developer with a blend of web and database application development skills. You will need to have a rigorous and conscientious approach to your work and enjoy overcoming challenging technical problems. You will need to have a strong grasp of the technologies that you have used so far and also be able to demonstrate to us that you can work independently in an organized manner.

If you like dealing directly with end-users, enjoy having autonomy over your work and prefer a diverse technical workload then you will like what we have to offer.

This position will likely be full-time but there is room for negotiation on this. There is also some scope for remote working for the right candidate. Remuneration is dependent upon experience.

How to Apply

Send your resume and a covering letter to: careers@softwaremodelers.com. (All inquiries will be treated with the utmost confidentiality and no employer or reference will be contacted without your consent.) We would like to thank all candidates for their interest but regret that we will only contact those that we wish to interview.
