Web and Application Engineers

Rewards Den in Victoria, BC

 

We are looking for A+ software engineers who are ready for a career defining opportunity. We are looking for individuals with strong backgrounds and interest in developing dynamic high volume mobile and web applications and analytics platforms. Ideal candidates are passionate and experienced Ruby developer, or very talented in Python and/or Java with strong PHP JavaScript, HTML5 and CSS skills.

Requirements

• Candidate must either have two years or more of PHP experience, or must have developed a significant web and or server applications - either in terms of scale or in terms of complexity

• Candidate should be able to work independently, taking ownership of major projects

Technology Experience

• PHP, Ruby/Rails, Python, Java

• JavaScript, HTML5, JSON, XML, etc

• Apache, Node.js

• AWS, EC3, S3, CloudFront/CDNs

• Linux, Ubuntu/Debian

Challenges we are tackling

• Intuitive UI/UX, HTML5/JS development for web and mobile

• Services that scale to millions and millions of requests per day

• Big data storage, search, processing and visualization

• Rock-solid platform specific SDKs, APIs

• Automated test systems, at scale, in the cloud

Responsibilities

• Develop for all aspects of the front-end and web application technology stack

• Develop functional and elegant user interfaces

• Create speed, improve performance and maintain stability within the development lifecycle

• Develop software and technology solutions, solve hard problems and answer to the technical needs of both the product and development teams

• Be a team player with a winning attitude

 

How to Apply

If you or someone you know is interested in joining the team, please contact us at: jobs@rewardsden.com
